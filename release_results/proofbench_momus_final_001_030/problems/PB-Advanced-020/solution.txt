**The Architect's Log (Hemingway Style):**
We examined the sequence for all positive integer pairs. We simplified the greatest common divisor. If the integers were equal, only the simplest pair, one and one, stayed constant. For all other equal pairs, the sequence grew without end. We then looked at unequal pairs. We assumed the sequence stopped moving. This assumption trapped the limit under a low ceiling. We then used powers and totients to find a term that broke through that ceiling. The ceiling and the term could not exist together. The sequence never stops moving for those pairs. Only the first pair is still.

**The Final Blueprint (Reviewer-Ready):**

**Problem:**
For given positive integers $x$ and $y$, let $(a_n)_{n \ge 1}$ be defined as
$$a_n = \gcd \left( x^n +y , \, (y-x)\left(\sum_{i=0}^{n-1} y^i x^{n-i-1} - 1\right) \right)$$
Find all pairs $(x,y)$ such that $\lim_{n \to \infty} a_n$ exists.

**Proof:**
A sequence of integers $(a_n)$ has a limit if and only if it is eventually constant. Let $x$ and $y$ be positive integers.

**Case 1: $x = y$**
If $x = y$, the second term of the $\gcd$ is $0 \cdot (\sum y^i x^{n-i-1} - 1) = 0$. The sequence becomes:
$$a_n = \gcd(x^n + x, 0) = x^n + x$$
For $(a_n)$ to be eventually constant, we require $a_{n+1} = a_n$ for all $n \ge N$:
$$x^{n+1} + x = x^n + x \implies x^{n+1} = x^n$$
Since $x$ is a positive integer, this equation is satisfied if and only if $x = 1$. Thus, $(x, y) = (1, 1)$ is a solution. For this pair, $a_n = 1^n + 1 = 2$ for all $n$, so the limit exists and is 2.

**Case 2: $x \neq y$**
Using the geometric series identity $\sum_{i=0}^{n-1} y^i x^{n-i-1} = \frac{y^n - x^n}{y - x}$ for $x \neq y$, the sequence is:
$$a_n = \gcd(x^n + y, (y - x) \left( \frac{y^n - x^n}{y - x} - 1 \right)) = \gcd(x^n + y, y^n - x^n - y + x)$$
Using the property $\gcd(A, B) = \gcd(A, A + B)$:
$$a_n = \gcd(x^n + y, (y^n - x^n - y + x) + (x^n + y)) = \gcd(x^n + y, y^n + x)$$
Suppose the limit exists, so $a_n = L$ for all $n \ge N$. Since $L$ divides both $x^n + y$ and $x^{n+1} + y$:
$$L \mid [x(x^n + y) - (x^{n+1} + y)] = xy - y = y(x - 1)$$
Similarly, by symmetry, $L \mid x(y - 1)$.

**Subcase 2a: $x = 1, y > 1$** (or $y = 1, x > 1$)
If $x = 1$, then $a_n = \gcd(1 + y, y^n + 1)$. 
If $n$ is odd, $y + 1$ divides $y^n + 1$, so $a_n = y + 1$. 
If $n$ is even, $y^n + 1 = (y^n - 1) + 2$. Since $y + 1$ divides $y^n - 1$ for even $n$, we have $a_n = \gcd(y + 1, 2)$.
For the limit to exist, we must have $y + 1 = \gcd(y + 1, 2)$, which implies $y + 1 \le 2$, so $y \le 1$. This contradicts $y > 1$.

**Subcase 2b: $x > 1, y > 1, x \neq y$**
If the limit $L$ exists, the condition $L \mid y(x - 1)$ implies $L \le y(x - 1) = xy - y$.
Define $m = xy + 1$. Since $x, y > 1$, $m \ge 5$. Note that $\gcd(x, m) = \gcd(x, xy + 1) = 1$.
By Euler's Totient Theorem, $x^{\phi(m)} \equiv 1 \pmod m$. Let $n = k\phi(m) - 1$ for some $k$ such that $n \ge N$.
For this index $n$, we have $x^{n+1} \equiv 1 \pmod m$, so $x^n \equiv x^{-1} \pmod m$.
Furthermore, $xy \equiv -1 \pmod m$ implies $y \equiv -x^{-1} \pmod m$.
Substituting these into $x^n + y$:
$$x^n + y \equiv x^{-1} + (-x^{-1}) = 0 \pmod m$$
Similarly, $y^n + x \equiv 0 \pmod m$. Thus $m \mid \gcd(x^n + y, y^n + x) = a_n$.
If $a_n = L$, then $L \ge m = xy + 1$.
Combining our bounds: $xy + 1 \le L \le xy - y$, which implies $1 \le -y$.
This is a contradiction for positive integer $y$.

Thus, no pairs $(x, y)$ with $x \neq y$ satisfy the condition except for the cases already excluded. The only pair is $(1, 1)$.

**The final answer is $(1, 1)$.**
